### Addon load.

1) Addon loads without error.
    - Steps:
        1) Load game with addon loaded.
        2) Load character in game with addon enabled.
        3) Bring up Options.
        4) Select Addons tab.
    - Expected Result:
        - No errors should occur during loading.
        - Addon should appear in options and display in the correct localizataion

2) Addon reloads without error.
    - Steps:
        1. Load game and character with addon enabled
        2. use console command `/reload'
    - Expected Result:
        - Reload should complete with no errors.
