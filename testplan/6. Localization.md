### Localization

Notes:
- Test should be repeated for each locale/language supported by the addon.

Prerequisites:
- Game is closed

Tests:
1) Settings page is localized.
    - Steps:
      1) Change the localization to different language in the launcher game settings.
      2) Start the game and load a character with the addon loaded.
      3) Open the settings page for the addon.
    - Expected Result:
      1) Addon page should have labels in the correct localization (credits are not localized).
      2) Existing saved values for eating and well-fed text should be preserved.


2) Default values are localized.
   - Steps:
     1) Delete the SavedVariabels file for the addon.
     2) Change the localization to different language in the launcher game settings.
     3) Start the game and load a character with the addon loaded.
     4) Open the settings page for the addon.
   - Expected Result:
     1) Addon page should have labels in the correct localization (credits are not localized).
     2) Saved values should be the default values fo that localization.
