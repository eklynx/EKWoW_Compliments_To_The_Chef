### Basic usage - combat

Notes:
- This test is easiest with a second character in group to get character in combat while eating. 

Prerequisites:
- Character is in game with addon loaded.
- Character is in an instance zone where they can be put in combat.

Tests:
1) When in-combat, character should no longer say anything. Everything shoul function normally when again out of combat.
    - Steps:
      1) Eat food that produces the well fed status effect.
      2) After starting to eat but before well fed buff is applied, get in to combat.
      3) Exit combat and eat food that produces the well fed status effect, staying out of combat. 
    - Expected Result:
      1) Character should say the initial 'Eating' message.
      2) Character should not say anything else for the initial food duration.
      3) The second food eating should have the character say both the eating and well fed messages as normal.
      4) No errors should be thrown.
