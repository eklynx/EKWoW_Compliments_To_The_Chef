### Settings

Prerequisites:
- Character is in game with addon loaded.
- Character is in an instance zone

Tests:
1) Changed settings are saved
    - Steps:
      1) Open settings to the addon settings section.
      2) Change both eating and well-fed messages.
      3) Click the save button.
      4) Exit settings and re-open them to the same location. Verify the saved settings are preserved.
      5) Exit settings and eat food that produces the well-fed buff.
    - Expected Result:
      1) Settings page loads without error showing the existing settings.
      2) Saved settings are preserved after exiting and re-opening settings.
      3) When eating, both the eating and well-fed messages are the newly saved messages.


2) Changed settings are reverted
    - Steps:
      1) Open settings to the addon settings section.
      2) Change both eating and well-fed messages.
      3) Click the revert button. Verify the settings are reverted to the previous values.
      4) Exit settings and re-open them to the same location. Verify the previous settings are preserved
    - Expected Result:
      1) Settings page loads without error showing the existing settings.
      2) Saved settings are preserved after exiting and re-opening settings.


3) Changed settings are not saved when closed.
    - Steps:
      1) Open settings to the addon settings section.
      2) Change both eating and well-fed messages.
      4) Exit settings without clicking save or revet.
      5) Re-open settings to the same location. Verify the previous settings are preserved
    - Expected Result:
      1) Settings page loads without error showing the existing settings.
      2) Saved settings are preserved after exiting and re-opening settings.


4) Settings are saved when saved followed by revert.
    - Steps:
      1) Open settings to the addon settings section.
      2) Change both eating and well-fed messages.
      3) Click the save button.
      4) Click the revert button. Verify settings are kept to the new values.
      5) Exit settings and re-open them to the same location. Verify the saved settings are preserved.
    - Expected Result:
      1) Settings page loads without error showing the existing settings.
      2) Saved settings are preserved after pressing the revert button.
      2) Saved settings are preserved after exiting and re-opening settings.


5) Settings are preserved between game sessions.
    - Steps:
      1) Open settings to the addon settings section.
      2) Change both eating and well-fed messages.
      3) Exit the game entirely.
      4) Re-open the game and verify the settings are preserved.
    - Expected Result:
      1) Saved settings are preserved after exiting and re-starting the game.

6) Default values are used when settings are cleared.
    - Steps:
      1) Open settings to the addon settings section.
      2) Change both eating and well-fed messages.
      3) Click the save button.
      4) Exit the game entirely.
      5) Delete the SavedVariabels file for the addon.
      6) Re-open the game and go to the settings page for the addon.
    - Expected Result:
      1) Eating and Well-Fed messages are set to the default values after settings are cleared.
